Day 19: JavaScript Intro + Variables + Output Statements (2025-04-07)

Practice Tasks

Create two variables:

let studentName = "Ali Khan"
let course = "JavaScript"

Then print both variables in the console.

Declare a variable:

let city = "Hyderabad"

Show an alert like this:

alert("Ali Khan lives in " + city)

Declare variables:

let fruit = "Mango"
let quantity = 5

Use console.log() to print:

console.log("I bought " + quantity + " " + fruit + "es")

Declare your name in a variable and show an alert saying:

alert("Welcome, " + yourName + "!")

Create a confirm popup and store result:

let result = confirm("Are you sure you want to continue?")
console.log(result)

Declare:

let book = "JavaScript Basics"

Log the message:

console.log("The book I am reading is " + book)

Set the body content using:

document.body.innerHTML = "<h2>My name is Abrar</h2>"

Declare:

let title = "Student Portal"

Then output to browser:

document.body.innerHTML = "<h1>" + title + "</h1>"

Ask for confirmation and show a message based on the response:

let proceed = confirm("Do you want to register for the course?")
if(proceed) { alert("Thank you for registering!") }

Declare:

let name = "Sarah"
let pet = "Cat"
let age = 25

Then log:

console.log(name + " is " + age + " years old and has a " + pet)
